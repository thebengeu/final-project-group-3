Overview

This operation downloads a playlist and all its constituent chunks, while at all times maintaining the invariant that an incomplete playlist should always be valid according to the HLS standard.

Parameters

The recording id that is being downloaded.

playlistURL

The source URL to download.

dir

The destination directory. Both the downloaded playlist and media chunks are stored in the same directory.

A delegate that can receive callbacks from this operation. Pass nil if callbacks are not required.
