Overview

A bidirectional mapping of recording id to HLSNetServicePathChunkCountTuple. The mapping is keyed on recording id, and an IP Address.

Tasks

  • – init

    Creates a new HLSDiscoveredRecordings.
  • – addDiscoveredRecordingId:onServiceNamed:tuple:

    Adds a mapping of recording id and IP Address, and stores associated data in a HLSNetServicePathChunkCountTuple.
  • – removeDiscoveredFromServiceNamed:

    Removes a mapping of recording id and IP Address by IP Address. Associated data is also deleted.
  • – netServicesWithRecording:

    Returns an NSMutableArray of NSNetServices which correspond to a recording id.
